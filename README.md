## Conversor de Moneda

---

### Descripción del Proyecto

Este es un conversor de moneda simple y eficiente desarrollado en **Java**. Permite a los usuarios convertir una cantidad específica de una moneda de origen a una moneda de destino, utilizando **tasas de cambio actualizadas** [2025].

---

### Características

* **Conversión Rápida**: Convierte entre una amplia gama de divisas.
* **Actualización de Tasas**: Utiliza una API externa para obtener las tasas de cambio más recientes.
* **Fácil de Usar**: Ejecución directa por línea de comandos.

---

### Uso

Debes crear una cuenta en https://www.exchangerate-api.com/ para obtener una apikey y agregarla en la parte de "resources" en una nueva fila llamada "config.properties"
Dentro solo debes agregar la linea:
<br>apikey=tukeyobtenida
